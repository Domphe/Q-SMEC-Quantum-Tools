# Open API Catalog (trusted, 2025-validated)

Government and standards:
- U.S. Census Bureau API — https://api.census.gov/data.html
- BEA (Bureau of Economic Analysis) — https://apps.bea.gov/API/signup/index.cfm
- BLS Public Data API — https://www.bls.gov/developers/
- EIA Open Data — https://www.eia.gov/opendata/
- DOE OSTI / Data Explorer — https://www.osti.gov/opendata/
- EPA Data — https://www.epa.gov/data
- NOAA Climate Data — https://www.ncdc.noaa.gov/cdo-web/webservices/v2
- NIST CISA/NVD (vuln & product) — https://nvd.nist.gov/developers
- NIST CSF / 800-53 refs (docs, no API) — https://csrc.nist.gov/
- CMS / HHS Protect (selected exports) — https://healthdata.gov/
- FDA open data — https://open.fda.gov/
- NIH PubMed / E-utilities — https://www.ncbi.nlm.nih.gov/books/NBK25501/
- SAM.gov Exports (opportunities) — https://open.gsa.gov/api/
- FERC / OpenEnergyData — https://www.ferc.gov/open-data
- USASpending — https://api.usaspending.gov/
- DOT BTS — https://www.bts.gov/bts-open-data
- DHS CISA (advisories, no single API) — https://www.cisa.gov/

Academic and repositories:
- Crossref — https://api.crossref.org/
- OpenAlex — https://docs.openalex.org/
- OSF — https://developer.osf.io/
- Zenodo — https://developers.zenodo.org/

Energy, water, environment (state):
- Arizona Open Data — https://azgeo-data-hub-azdot.hub.arcgis.com/ and https://data.az.gov/

Note: Some endpoints require tokens. Check `.env`.
